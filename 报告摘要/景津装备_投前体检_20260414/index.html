    景津装备_投前体检¶

    压滤机龙头周期底部，等待锂电β修复

    基本信息¶

    • 公司：景津装备（603279）
    • 行业：压滤机、环保设备、通用设备
    • 报告日期：2026-04-14
    • 数据时效：2025年10月–2026年4月

    核心结论¶

    景津装备是压滤机行业绝对龙头，国内市占率40%+。2025年前三季度营收42.00亿元，同比下降10.70%；归母净利润4.35亿元，同比下降34.53%，主要受新能源（锂电）资本开支下滑和产品降价影响。当前PE 17.02倍，处历史低位，股息率4.93%提供安全边际。

    多空核心分歧：短期业绩承压 vs 下游景气拐点。空头逻辑核心是新能源需求持续下行+价格战压缩利润；多头逻辑是锂电β修复+矿业资本开支提升+成套装备放量+出海加速。

    投资结论：当前价位（18.27元）估值合理偏低，适合分批布局，但需等待业绩拐点确认。

    关键变量¶

    宏观变量¶

    • 宏观变量/碳酸锂价格周期：正面影响，碳酸锂价格从6.14万元/吨涨至9.77万元/吨（+59.1%）

    主题变量¶

    • 主题变量/锂电产业链复苏：正面影响，新能源业务占比13.39%，正极材料龙头扩产计划密集发布

    行业变量¶

    • 行业变量/矿山设备/压滤机行业竞争格局：负面影响，价格竞争加剧，毛利率下降4.59pct
    • 行业变量/矿山设备/矿物加工设备需求：正面影响，铜金价格高位刺激资本开支

    公司变量¶

    • 公司变量/景津装备/成套装备产能：正面影响，三期产能2026年1月开工
    • 公司变量/景津装备/海外业务：正面影响，海外毛利率56%（较国内+28pct）
    • 公司变量/景津装备/应收账款：需关注，经营现金流当前健康

    估值分析¶

    PE（静） 12.41倍 历史低位 低估

    股息率 4.93% 历史高位 有吸引力

    基于2026年预测（7亿元净利润中枢）： - 合理估值（15倍PE）：105亿元 → 上涨空间有限 - 乐观估值（20倍PE）：140亿元 → 上涨空间33% - 悲观估值（12倍PE）：84亿元 → 下跌空间20%

    风险因素¶

    1. 新能源需求持续下行：新能源收入占比从2023年26.56%降至2025H1的13.39%
    2. 行业价格竞争加剧：2025Q1-3毛利率24.47%，同比下降4.59pct
    3. 订单能见度低：压滤机是资本开支后周期设备，需求传导需要时间
    4. 应收账款风险：需关注（当前经营现金流良好）

    业绩拐点确认信号¶

    1. 订单层面：单月新签订单同比转正
    2. 营收层面：季度营收同比降幅收窄至5%以内
    3. 毛利率层面：季度毛利率企稳回升
    4. 锂电产业链：头部企业扩产项目正式落地
