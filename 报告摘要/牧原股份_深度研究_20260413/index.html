    牧原股份深度研究¶

    报告类型：深度分析 | 原始资料：深度报告/牧原股份_商业模式评估_20260411.md + 牧原股份_正负面预期_20260411.md | 编译时间：2026-04-13

    报告概览¶

    本报告对牧原股份进行商业模式评估和正负面预期分析，判断公司投资价值和时机。

    核心结论¶

    商业模式评级：A级¶

    核心优势： - 全球成本最低的生猪养殖企业（完全成本11.6-12元/公斤） - 全产业链垂直整合，协同效应显著 - 规模全球第一，护城河深厚

    主要短板： - 盈利高度依赖猪价，周期性明显 - 短期债务压力较大 - 分红率过高削弱现金储备

    投资建议：逢低布局，等待周期拐点¶

    • 当前估值处于历史低位（PE-TTM约12-15倍）
    • 2026年下半年猪周期有望触底回升
    • 牧原作为行业龙头将充分受益周期反转

    成本优势来源¶

    成本项 牧原优势 贡献度

    二元轮回育种 种猪100%自供 约0.3元/公斤
    智能化养殖 人均效率提升、死亡率下降 约0.5元/公斤

    正面预期¶

    1. 成本优势持续扩大¶

    • 2026年目标降至11.5元/公斤以下
    • 部分优秀场线已突破11元/公斤
    • 较行业平均低1.5-3元/公斤

    2. 屠宰业务里程碑式突破¶

    • 首次实现年度盈利
    • 产能利用率98.8%
    • 毛利率同比提升1.65个百分点至2.67%

    3. 产能去化加速¶

    • 能繁母猪从362万头降至313万头（-13.5%）
    • 停建124个在建猪场（约1,200万头产能）
    • 2026年二季度加速去化，三季度接近平衡

    4. 国际化战略推进¶

    • 港股上市募资约104.6亿港元
    • 60%用于海外业务拓展
    • 越南项目2026年落地自有产能

    负面预期¶

    1. 猪价持续低迷¶

    • 2026年3月跌破10元/公斤，创近8年新低
    • 市场已将极度悲观预期纳入定价
    • PE-TTM约12-15倍，处于历史20%分位

    2. 高负债压力¶

    • 有息负债832亿元，短期债务482.7亿元
    • 流动比率0.78、速动比率0.25，偏低
    • 港股上市补充流动性，短期风险可控

    3. 盈利高度集中¶

    • 养殖业务营收占比97%，业务结构单一
    • 猪价每变动1元/公斤，影响净利润约92亿元
    • 屠宰业务毛利率仅2.67%，对冲作用有限

    4. 分红政策争议¶

    • 2025年分红率48.03%，超过40%目标
    • 周期底部高分红消耗留存收益
    • 实控人夫妇持股54.91%，获益约40亿元

    3. 周期股波动性大，需承受短期浮亏
