    多空逻辑¶

    多头逻辑（4条）¶

    1. 光气资源壁垒：生产资质稀缺，国内少数企业拥有
    2. 产业链一体化：码头、热电、污水处理配套自供，成本优势突出
    3. 政策驱动整合："一证一品"政策推动产能出清，龙头受益
    4. 高股息+低估值：近三年累计派现14-16亿元，PE处于历史低位

    空头逻辑（4条）¶

    1. 行业周期下行：农药行业底部，产品价格内卷严重
    2. 盈利能力下滑：ROE从30.70%降至8.08%，毛利率从37%降至30%
    3. 印度产能冲击：全球供应过剩，延长价格底部周期
    4. 新业务不确定：医药中间体、新材料拓展处于起步阶段
