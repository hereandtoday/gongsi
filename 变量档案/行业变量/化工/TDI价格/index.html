    TDI价格¶

    行业变量（化工） | 影响级别：高 | 影响方向：正面/负面 | 最后更新：2026-04-13

    变量定义¶

    TDI（甲苯二异氰酸酯）主要用于聚氨酯软泡生产，下游应用包括家具（沙发、床垫）、汽车座椅、包装材料等领域。TDI与MDI并称为聚氨酯行业的两大核心产品。

    当前状态（2026年4月更新）¶

    价格区间： - TDI价格：约1.6万元/吨（较2022年低点上涨约23%）

    库存水平： - TDI社会库存仅为往年正常水平的50-60%，处于极低位置

    产能格局： | 厂商 | 产能（万吨/年） | 全球市占率 | |------|---------------|-----------| | 万华化学 | 147 | 40%+ | | 巴斯夫 | 80 | 22% | | 科思创 | 60 | 16% | | 其他 | 80 | 22% |

    供需变化（2025年4月-2026年4月）¶

    供给端：海外加速退出，国内主导地位确立¶

    海外产能持续收缩： | 地区/企业 | 变化情况 | |----------|---------| | 阿根廷 | 淘汰3万吨产能 | | 日本三井 | 产能从12万吨缩减至5万吨 | | 日本东曹 | 2.5万吨装置关停 | | 科思创德国多马根 | 2025年七八月因不可抗力停车，复产推迟 | | 沙特 | 20多万吨产能因地缘冲突受阻 | | 伊朗 | 4万吨产能受阻 |

    全球停产规模： - 截至2026年1月，全球约86万吨TDI装置处于停车检修状态 - 占总产能近25%

    国内产能稳步扩张： - 2024年国内TDI总产能152万吨，产量134万吨（同比+18%） - 主要增量：万华福建一期、甘肃银光装置投产 - 华鲁恒升计划2025-2027年建设30万吨TDI产能 - 中国TDI产能占全球比重预计从54.7%进一步提升

    需求端：内需企稳，出口强劲¶

    内需边际改善： - TDI主要用于软体家具（海绵、床垫、沙发）、汽车座椅 - 与地产竣工端关联紧密，过去几年受地产低迷拖累 - 2025年"以旧换新"政策扩大至家装及软体家具，内需有效托底 - 预计2025年中国TDI消费量约91.4万吨

    出口成为核心驱动力： - 2024年TDI出口量约37万吨 - 2025年TDI累计出口量约56万吨，同比+52% - 出口量占产量比重超35% - 中东局势紧张推动周边国家增加从中国进口

    价格周期特征¶

    阶段 时间 价格区间 驱动因素
    高景气 2021年 2.0-2.5万元/吨 疫情后需求爆发

    复苏初期 2025Q4至今 1.5-1.7万元/吨 海外产能退出+出口强劲

    业绩传导¶

    TDI价格每上涨1000元/吨，对万华化学年化净利润影响约3-5亿元。

    关键驱动因素¶

    正面因素： 1. 海外产能加速退出（全球约86万吨停车） 2. 库存极低（仅为正常水平50-60%） 3. 出口强劲增长（2025年同比+52%） 4. 政策托底内需（以旧换新）

    负面因素： 1. 地产后周期拖累 2. 国内产能扩张压力（华鲁恒升30万吨规划）
