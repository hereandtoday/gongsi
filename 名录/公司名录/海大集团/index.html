    海大集团（002311）¶

    公司概况¶

    行业定位：全球饲料龙头，"饲料+种苗+动保+服务"全产业链协同

    核心数据（2024年）： - 营收：1,146亿元（-1.31%） - 净利润：45.04亿元（+64.30%） - ROE：20.88%（+6.35pct） - 净利率：4.54%（+1.80pct） - 饲料销量：2,652万吨（全球第一，+8.7%） - 水产饲料市占率：全球40%+，国内18%

    商业模式评级：A级（优秀）

    业务构成： | 业务 | 营收占比 | 毛利率 | |------|---------|--------| | 饲料销售 | 79.05% | 9.74% | | 农产品销售 | 16.11% | 18.28% | | 贸易业务 | 3.69% | 0.65% | | 动保产品 | 0.89% | 56.48% |

    投资结论摘要¶

    综合评级：A级（值得配置，价值凸显）

    商业模式评级：A级（优秀）—— 具备长期投资价值的优质核心资产

    估值水平：PE约17倍，近3年13%分位，历史低位

    核心财务数据（2024年）： - 营收：1,146亿元（-1.31%） - 净利润：45.04亿元（+64.30%） - ROE：20.88%（+6.35pct） - 净利率：4.54%（+1.80pct） - 饲料销量：2,652万吨（全球第一）

    核心催化剂： 1. 海外业务+40%持续验证 2. 港股分拆上市（2026年中） 3. 市占率提升（8.41%→9.27%） 4. 回购10亿元+分红率提升至50%+

    核心风险： 1. 生猪养殖周期底部（影响<5%，已脱敏） 2. 原材料价格反弹（有对冲机制） 3. 分拆上市短期股价承压 4. 国内饲料行业增速放缓 5. 海外扩张的地缘政治风险

    投资建议：当前估值具备极高安全边际，建议逢低布局，中长期持有

    目标价：63.33元（机构共识），上涨空间约30%

    正面催化剂¶

    1. 饲料外销高增长：2025年上半年外销增长25%
    2. 港股分拆加速：海大国际赴港上市，拓宽融资渠道
    3. 国际化战略加速：海外销量目标2030年720万吨
    4. Q1业绩高增长：归母净利润12-13亿元

    负面风险¶

    1. 养殖后周期压力：猪周期下行可能影响饲料需求
    2. 原材料成本压力：豆粕、玉米等价格波动
    3. 行业竞争加剧：新希望、通威等持续竞争
