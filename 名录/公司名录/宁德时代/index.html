    宁德时代（300750）¶

    正面催化剂¶

    1. 全球龙头地位稳固：动力电池市占率39.2%，连续9年第一
    2. 一季报超预期：Q1净利润+48.52%
    3. 储能第二增长曲线爆发：Q1储能出货+108%
    4. 4月21日Tech Day催化：凝聚态电池量产、超充迭代

    负面风险¶

    1. 估值分位偏高：PE 87%分位，接近警戒线
    2. 原材料成本压力：碳酸锂价格反弹
    3. 竞争加剧：二线厂商追赶+车企自建电池厂

    核心竞争力¶

    全球双料冠军¶

    业务 全球市占率 排名
    动力电池 39.2% 连续9年第一
    储能电池 30.4% 连续5年第一

    四重护城河¶

    1. 成本壁垒：规模效应+极限制造+锂资源自给率>30%
    2. 技术壁垒：54,538项专利，麒麟/神行/钠离子全球领先
    3. 转换成本：深度绑定+定制化+换电生态
    4. 规模壁垒：全球最大产能772GWh

    商业模式特点¶

    • 强现金流机器：经营现金流/净利润>184%
    • 全球化布局：德国工厂已盈利，匈牙利工厂投产中
    • 储能第二曲线：AI数据中心配储打开天花板

    增长驱动力分析¶

    1. 市占率提升：动力电池全球份额持续提升
    2. 储能爆发：Q1储能出货+108%，第二增长曲线
    3. 技术迭代：Tech Day发布凝聚态电池、超充技术
    4. 全球化：匈牙利工厂量产，海外份额提升
