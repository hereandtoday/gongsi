    兖矿能源¶

    商业模式评估摘要（2026-04-16）¶

    主营业务 B+ 煤炭+煤化工五大产业
    产业链位置 B+ 国内外煤炭资源布局
    财务表现 B 2025年净利润84亿元（-43.61%）

    成长性 B+ 产量增长对冲煤价下行

    核心财务数据（2025年）： - 营业收入：1449.33亿元（-7.49%） - 归母净利润：83.81亿元（-43.61%） - 商品煤产量：1.82亿吨（+28.2%）

    报告摘要： - 报告摘要/兖矿能源_商业模式评估_20260416 - 深度报告/兖矿能源_商业模式评估报告_1776342123121_8_9n95

    基本信息¶

    • 股票代码：600188.SH / 01171.HK
    • 行业：煤炭开采加工
    • 核心业务：煤炭开采+煤化工+高端装备+智慧物流
    • 公司评级：B+级（周期龙头）

    核心竞争优势¶

    1. 资源壁垒深厚：466亿吨煤炭资源量，境内外五大基地
    2. 煤化一体协同：自产煤炭为化工提供低成本原料，平滑煤价波动
    3. 定制化服务：72%定制化产品占比，溢价80元/吨
    4. 产能扩张：2026年目标1.86-1.9亿吨，"十五五"目标3亿吨

    核心催化剂¶

    1. 煤价反弹：秦皇岛5500大卡突破700元/吨
    2. 产能释放：西北矿业并表+新矿投产
    3. 化工爆发：甲醇价格维持3000元/吨以上
    4. 资产转让：鑫泰公司30.5亿元出售完成

    主要风险¶

    1. 煤价持续下跌：跌破400-450元/吨，接近盈亏平衡
    2. 债务压力：有息负债1494亿元，现金流覆盖趋弱
    3. 成本劣势：吨煤成本321元/吨，较陕煤、神华高约100元
    4. 澳洲资产波动：兖煤澳洲利润下滑60%
